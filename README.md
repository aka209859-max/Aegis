## ✨ 主要機能（実装済み）

### 1. 未払いリスクスコア算出
```
S_unpaid = 0.45 × K_legal + 0.35 × C_credit + 0.20 × P_process
```
- **K_legal**: 契約書の法的強度（業務委託契約書の有無）
- **C_credit**: クライアントの信用リスク（支払遅延履歴）
- **P_process**: 証拠保全・管理リスク（請求書・コミュニケーション記録）

### 2. デジタル遺産承継難易度算出
```
S_digital = Σ(V_i × 1/A_i) × 税制係数(0.8)
```
- **V_i**: デジタル資産価値（暗号資産・NFT・ドメイン等）
- **A_i**: アクセス難易度（パスワード管理状況）
- **税制係数**: 2026年税制改正による「時価80%評価」

### 3. XAI（説明可能なAI）実装
- 推論プロセスの完全可視化
- 各変数の影響度を数式で表示
- ユーザーがスコア算出根拠を確認可能

### 4. 処方箋（Prescription）UX
- **リスク60点以上**: 弁護士無料相談への緊急誘導
- **リスク60点未満**: FREENANCE即日払いサービスへの予防的誘導
- 「広告」ではなく「診断結果に基づく改善命令」として提示

### 5. 弁護士監修コラムの自動表示
- 各質問に連動して法的情報を表示
- 2026年法改正（電子帳簿保存法、フリーランス保護法）に完全対応
- 判例引用・法的根拠の明示

## 📊 データアーキテクチャ

### 処理フロー
1. **質問応答**: 8問の診断フォーム（一問一答形式）
2. **スコア算出**: JavaScript内での完全計算（サーバー送信なし）
3. **XAI表示**: 計算式と各変数の可視化
4. **処方箋生成**: リスクレベルに応じた改善命令

### データ管理
- **ストレージ**: 一切使用しない（完全揮発性）
- **プライバシー**: ブラウザ閉じた瞬間に全データ消失
- **GDPR準拠**: データ保存がないため、削除要求への対応不要
